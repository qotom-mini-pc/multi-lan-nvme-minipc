# Qotom Q31100G4 Multi-LAN + NVMe Mini PC

## Overview

This project documents a compact multi-LAN + NVMe platform for:

- NAS (storage)
- Router / firewall
- Homelab services

---

## Hardware

- 4 × Intel I226-V 2.5GbE LAN
- 4 × M.2 NVMe
- Intel Tiger Lake CPU

---

## Use Case

- NAS + router combined
- Multi-LAN network separation
- Edge computing node

---

## Keywords

mini PC NAS, multi LAN mini PC, Qotom Q31100G4, homelab server
